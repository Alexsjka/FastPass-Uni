# FastPass-Uni

## Descripción del Proyecto
Este repositorio contiene el desarrollo del **caso práctico**, donde se realizó el análisis del sistema utilizando **historias de usuario, requerimientos y diagramas** para representar el funcionamiento del sistema.

El objetivo del proyecto es documentar y estructurar la solución del problema planteado mediante herramientas de modelado y documentación.
